# 🌱 AgriSmart AI
### AI-Powered Smart Agriculture System

> An intelligent farming assistant that helps farmers with **Crop Recommendation**, **Plant Disease Detection**, **Weather Information**, **Market Price Prediction**, and an **AI Chatbot**.

---

## 🚀 Features

### 🌾 Crop Recommendation
- Predicts the best crop based on soil nutrients and environmental conditions.
- Machine Learning based recommendation system.

### 🍃 Plant Disease Detection
- Detects plant diseases from uploaded leaf images.
- CNN (Convolutional Neural Network) based image classification.

### 🌦 Weather Information
- Displays current weather conditions.
- Helps farmers plan agricultural activities.

### 📈 Market Price Prediction
- Predicts crop prices using Machine Learning.
- Helps farmers decide the best time to sell crops.

### 🤖 AI Farming Assistant
- AI chatbot for agriculture-related queries.
- Provides farming guidance and crop information.

# ⚙ Installation

Clone the repository

```bash
git clone https://github.com/itzpooja1144/AgriSmart-AI.git
```

Move inside the project

```bash
cd AgriSmart-AI
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python backend/app.py
```

Open browser

```
http://127.0.0.1:5000
```
